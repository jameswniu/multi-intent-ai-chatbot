# Example User Guide

Adding Users
To add a new user, open Settings > Users > Add New. Enter the user’s details and assign permissions.

Resetting Passwords
Select the user from the Users list and click “Reset Password.” The user will receive an email link.

Analytics Module
The Analytics module allows dashboard creation, KPI tracking, and report scheduling.

Contracts
Contract renewals are managed under Billing > Account Management. You can view module expiry dates and pricing there.
